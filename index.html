Commandes terminal Linux

Commandes Signification Description Exemple d'utilisation
ls list segment Liste un répertoire ls -a : liste tous les fichiers et répertoires
cd change directory Permet de naviguer dans les répertoires cd .. : permet de remonter au répertoire parent
mv move Permet de déplacer ou renomer des fichiers mv monFichier Rep/ : déplace mF dans Rep
cp copy Copie des fichiers cp monFichier Rep/ : copie mF dans Rep
rm remove Efface des fichiers rm Fichier : Efface Fichier du répertoire courant
mkdir make directory Crée un répertoire vide mkdir Photos : crée le répertoire photos
pwd print working directory Affiche le répertoire .
find rechercher Rechercher des fichiers find monfichier* : Recherche un fichier commençant par "monfichier"
grep global regular expression print Recherche une chaîne de caractères dans les fichiers grep -n montexte monfichier : Retourne toutes les lignes ainsi que leur numéro où montexte apparait dans monfichier
cat concatenate Affiche le contenu d'un fichier cat nomFichier : affiche le contenu de monFichier
more more Affiche un fichier page par page more -sf monFichier : Afficher monFichier page par page
less less Affiche un fichier permettant la navigation .
